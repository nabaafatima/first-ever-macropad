# Firmware
### I used KMK and it was my first time using it. thats why i had to do a lot of research for it to go well. Searched for all the possible shortcuts and found these the most helpful:
- key 1 -> copy
- key2 -> paste
- key 3 -> undo
- key 4 -> rdeo
